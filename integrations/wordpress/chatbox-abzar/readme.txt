ابزارک چت آنلاین ChatBox برای وردپرس — بدون کدنویسی.

== Description ==

این پلاگین ویجت ChatBox را روی سایت وردپرس شما بارگذاری می‌کند.

از پنل ChatBox (تنظیمات → ویجت) آدرس API و Workspace Slug را کپی و در تنظیمات پلاگین وارد کنید.

== Installation ==

1. پوشه `chatbox-abzar` را در `wp-content/plugins/` آپلود کنید (یا فایل zip پلاگین را از مخزن ChatBox بگیرید).
2. در وردپرس: افزونه‌ها → ابزارک ChatBox → فعال‌سازی.
3. تنظیمات → ابزارک ChatBox — API URL و Workspace Slug را وارد کنید.

== Frequently Asked Questions ==

= آدرس API چیست؟ =

همان آدرسی که داشبورد ChatBox روی آن اجرا می‌شود، مثلاً `https://api.yourdomain.com` (بدون اسلش انتهایی).

= ویجت روی کدام صفحات نمایش داده می‌شود؟ =

روی تمام صفحات عمومی (front-end) سایت.
